# MeuCasamento - Backend

Este repositório contém o backend de uma API para um sistema de gestão de festas de casamento.
O projeto teve origem em um trabalho acadêmico, no qual foram realizados o levantamento de requisitos, diagrama de classe e a modelagem do banco de dados (MER), mas não houve implementação prática.
